## Chi sono

Sviluppatore full-stack con base in Italia. Fondatore di [**Auriti Labs**](https://github.com/Auriti-Labs).

Mi occupo di **AI tooling**, **Generative Engine Optimization** e **web development** — dal backend alla produzione. Non scrivo codice per dimostrare qualcosa. Lo scrivo per risolvere problemi, automatizzare processi e costruire strumenti che funzionano davvero.

## Progetti — Auriti Labs

### [`geo-optimizer-skill`](https://github.com/Auriti-Labs/geo-optimizer-skill)
Toolkit completo per la **Generative Engine Optimization** — audit, ottimizzazione e visibilità nei motori di ricerca AI (ChatGPT, Perplexity, Claude, Gemini). Basato sulla ricerca Princeton KDD 2024.

### [`kiro-memory`](https://github.com/Auriti-Labs/kiro-memory)
Sistema di **memoria persistente cross-sessione** per CLI AI. TypeScript + SQLite + FTS5 + Express + MCP server + React web UI. Ogni sessione ricorda il contesto delle precedenti.

### [`kore-memory`](https://github.com/Auriti-Labs/kore-memory)
Il layer di memoria che **pensa come un umano**: ricorda ciò che conta, dimentica ciò che non serve, e non chiama mai casa. Python + FastAPI + SQLite + sentence-transformers. Decadimento Ebbinghaus + ricerca semantica.
